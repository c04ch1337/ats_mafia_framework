Quick Start Guide

Docs Home › Quick Start

Get your ATS MAFIA training environment running in just 5 minutes

5 minutes to complete

Prerequisites

Docker & Docker Compose

For containerized deployment

LLM API Key

OpenAI, Anthropic, or Google

8GB+ RAM

16GB recommended

Modern Browser

Chrome, Firefox, Safari, Edge

Configure Environment

Copy .env.example to .env and set the minimum required variables.

  • DATABASE_PASSWORD — required for Postgres
  • REDIS_PASSWORD — required for Redis
  • WEBHOOK_URL — required if using Personal Assistant telephony webhook
  • Provider credentials (if using phone integration):
    • TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_PHONE_NUMBER
    • or PLIVO_AUTH_ID, PLIVO_AUTH_TOKEN, PLIVO_PHONE_NUMBER
Ports and Services: As defined in docker-compose.yml — API: 8000, WebSocket: 8080, UI: 8501, Personal Assistant: 5000. These compose-defined ports take precedence even if .env.example contains API_PORT=5000.
1

Clone & Configure

First, clone the repository and set up your environment variables.

# Clone the repository git clone https://github.com/atsmafia/framework.git cd ats_mafia_framework # Copy environment template cp .env.example .env # Edit .env file with your settings nano .env
Pro Tip: At minimum, you need to set one LLM API key (OpenAI, Anthropic, or Google) and a database password.
Environment Configuration Guide

Launch Services

For troubleshooting, see the FAQ: Troubleshooting.

2

Launch Services

Start all required services using Docker Compose.

# Start all core services docker-compose up -d # Check service status docker-compose ps
What's Starting: API server (port 8000), WebSocket server (port 8080), Web UI (port 8501), PostgreSQL, and Redis.
Troubleshooting Docker Issues
3

Access the UI

Open your web browser and navigate to the ATS MAFIA dashboard.

# Open in your browser http://localhost:8501 # Or if running on remote host http://192.168.1.100:8501

You should see the "Don's Office" dashboard with system status indicators.

Open Dashboard
4

Create Your First Profile

Set up a training profile to define your role and capabilities.

  1. Navigate to the "Profiles" page
  2. Click "Add Profile"
  3. Choose a profile type (Red Team, Blue Team, or Social Engineer)
  4. Configure skill levels and specializations
  5. Save your profile
Recommendation: Start with a "Red Team Operator" profile for penetration testing training.
5

Launch Your First Training Session

Start your first training scenario to experience the ATS MAFIA system.

  1. Go to the "Scenarios" page
  2. Select "Basic Penetration Test"
  3. Choose your profile and set a budget (e.g., $10)
  4. Click "Launch Session"
  5. Monitor progress in the Situation Room
Budget Tip: Start with a small budget ($5-10) for your first session to understand cost tracking.

Verification Checklist

  • API Docs: http://localhost:8000/docs
  • WebSocket endpoint: ws://localhost:8080 — ws://localhost:8080
  • UI: http://localhost:8501

Personal Assistant (Optional)

If running the Personal Assistant, it is available on http://localhost:5000.

See the guide: docs/PERSONAL_ASSISTANT_DOCKER_GUIDE.md

Congratulations!

You've successfully set up ATS MAFIA and launched your first training session.
