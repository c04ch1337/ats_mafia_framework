Managing Profiles

Profiles define the persona, skills, and capabilities of your training agents. Each profile represents a different role in cybersecurity training.

Understanding Profile Types

Red Team Operator

Specializes in offensive security techniques including penetration testing, vulnerability assessment, and exploitation.

  • Network reconnaissance
  • Vulnerability scanning
  • Exploitation techniques
  • Post-exploitation activities

Blue Team Defender

Focuses on defensive security operations including incident response, threat hunting, and security monitoring.

  • Incident detection
  • Threat analysis
  • Security monitoring
  • Forensic analysis

Social Engineer

Expert in social engineering tactics including phishing, pretexting, and human intelligence gathering.

  • Phishing campaigns
  • Pretext development
  • Human psychology
  • OSINT gathering

Creating a New Profile

  1. Navigate to Profiles

    Click on the "Profiles" link in the main navigation to access the profile management interface.

  2. Click "Add Profile"

    Locate the "Add Profile" button in the top-right corner of the profiles page.

  3. Choose Profile Type

    Select from Red Team, Blue Team, or Social Engineer based on your training goals.

  4. Configure Basic Settings

    Set the profile name, description, and basic personality traits.

  5. Set Skill Levels

    Adjust skill levels for various capabilities (1-10 scale) to match your expertise.

  6. Save Profile

    Review your settings and click "Save" to create the profile.

Pro Tip: Start with lower skill levels (3-5) if you're new to cybersecurity training. You can always increase the difficulty later as you gain experience.

Profile Configuration Options

Setting Description Recommended Range
Skill Level Overall expertise in the profile domain 1-10 (Beginner to Expert)
Aggressiveness How bold the agent is in taking actions 1-10 (Cautious to Aggressive)
Creativity Ability to think outside standard procedures 1-10 (Methodical to Creative)

Creating and Managing Scenarios

Scenarios are the training exercises that define objectives, constraints, and success criteria for your training sessions.

Built-in Scenario Categories

Network Penetration Testing

Comprehensive network security assessments including reconnaissance, scanning, and exploitation.

Social Engineering

Human-focused security testing including phishing, pretexting, and influence techniques.

Incident Response

Blue Team scenarios focusing on detection, analysis, and response to security incidents.

Threat Hunting

Proactive threat detection and analysis scenarios for advanced security teams.

Vulnerability Assessment

Systematic identification and evaluation of security vulnerabilities.

Digital Forensics

Evidence collection and analysis scenarios for incident investigation.

Creating Custom Scenarios

  1. Define Objectives

    Clearly outline what the trainee should accomplish. Be specific about success criteria.

    Example Objective: "Gain administrative access to the target web server and exfiltrate the configuration file without triggering the intrusion detection system."
  2. Set Difficulty Level

    Choose from Beginner, Intermediate, Advanced, or Expert based on the target audience.

  3. Configure Environment

    Define the target systems, network topology, and available tools.

  4. Add Constraints

    Set time limits, budget constraints, and rules of engagement.

  5. Test and Validate

    Run through the scenario yourself to ensure it's solvable and properly configured.

Scenario Best Practices

Start Simple: Begin with straightforward objectives and gradually increase complexity as trainees progress.
Provide Clear Success Criteria: Trainees should know exactly what constitutes successful completion.
Avoid Ambiguity: Unclear objectives lead to frustration and reduced learning effectiveness.

Training Sessions

Training sessions are live exercises where profiles execute scenarios in real-time with cost tracking and performance monitoring.

Launching a Training Session

  1. Select Scenario

    Browse the available scenarios and select one that matches your training objectives.

  2. Choose Profile

    Select the profile that will execute the scenario. Ensure the profile type matches the scenario requirements.

  3. Set Budget

    Define a cost limit for the session. This helps control LLM API costs and encourages efficiency.

    Recommended Budgets:
    • Beginner scenarios: $5-10
    • Intermediate scenarios: $10-25
    • Advanced scenarios: $25-50
    • Expert scenarios: $50+
  4. Configure Session

    Set session name, description, and any additional parameters.

  5. Launch Session

    Click "Launch Session" to start the training exercise.

Monitoring Live Sessions

Once a session is launched, you can monitor it in real-time through the Situation Room:

Real-time Metrics

Watch progress indicators, success rates, and performance metrics update live.

Agent Communication

View the agent's thought process and decision-making in real-time.

Cost Tracking

Monitor session costs and get alerts when approaching budget limits.

Session Controls

Control Function When to Use
Pause Temporarily halt the session To analyze progress or intervene
Resume Continue a paused session Ready to proceed after analysis
Stop Terminate the session Emergency stop or completion
Intervene Provide guidance to the agent Agent is stuck or off-track
Session Persistence: Stopped sessions cannot be resumed. Use Pause if you plan to continue later.

Monitoring Training Sessions

Effective monitoring is crucial for understanding agent behavior and optimizing training outcomes.

The Situation Room

The Situation Room is your command center for monitoring active training sessions. It provides real-time visibility into agent activities and system performance.

Key Monitoring Components

Performance Dashboard

Real-time display of success rates, completion percentages, and efficiency metrics.

Activity Feed

Chronological log of all agent actions, decisions, and system events.

Agent Thoughts

Insight into the agent's reasoning process and decision-making logic.

Cost Monitor

Live cost tracking with budget alerts and spending analysis.

Time Tracking

Session duration, task completion times, and efficiency metrics.

Alert System

Notifications for important events, budget warnings, and system issues.

Understanding Agent Behavior

When monitoring sessions, pay attention to these behavioral indicators:

Decision Patterns: How the agent approaches problems and makes choices under different conditions.
Adaptability: How well the agent adjusts when faced with unexpected obstacles or failures.
Efficiency: The ratio of successful actions to total attempts, indicating resource optimization.

Intervention Guidelines

Knowing when to intervene is key to effective training:

Intervene When: The agent is stuck in a loop, making clearly incorrect decisions, or has been inactive for an extended period.
Observe When: The agent is learning from mistakes, trying different approaches, or making reasonable progress.

Cost Management

Effective cost management ensures you get maximum training value while controlling expenses.

Understanding Cost Components

Training session costs primarily come from LLM API usage:

LLM Processing

Cost of AI model inference for agent decision-making and analysis.

Token Usage

Cost based on input/output tokens processed by language models.

Session Duration

Longer sessions generally incur higher costs due to more AI interactions.

Cost Optimization Strategies

Set Appropriate Budgets: Start with smaller budgets ($5-10) and increase as you understand cost patterns.
Choose Efficient Models: Use cost-effective models for routine tasks and premium models only when necessary.
Monitor in Real-Time: Keep an eye on the cost monitor during sessions to prevent overspending.
Use Budget Alerts: Set alerts at 75% and 90% of your budget to get warnings before hitting limits.

Budget Planning

Plan your training budget based on these factors:

Training Level Recommended Budget/Session Weekly Budget Monthly Budget
Beginner $5-10 $20-40 $80-160
Intermediate $10-25 $40-100 $160-400
Advanced $25-50 $100-200 $400-800
Expert $50-100 $200-400 $800-1600

Cost Monitoring Tools

The system provides several tools to help you manage costs:

  • Real-time Cost Tracker: Live cost display during sessions
  • Budget Alerts: Automatic notifications at predefined thresholds
  • Cost Analytics: Historical cost analysis and trends
  • ROI Calculator: Measure training value against costs
  • Optimization Recommendations: AI-powered cost-saving suggestions
Cost Awareness: Always monitor costs during sessions, especially when using advanced AI models or running complex scenarios.
